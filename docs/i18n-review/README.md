## How to review (≈1 hour each)

1. Open the CSV in Google Sheets / Excel.
2. Column **english** = source of truth meaning.
3. Column **current_draft** = what is in the app today (may still be English).
4. Fill **corrected_by_native** with the approved string.
5. Return the sheet to Verry — we paste into `artifacts/api-server/public/dashboard/i18n.js` and flip `review: "ok"`.

Do **not** invent restaurant money/health advice in these strings — they are UI chrome only.
